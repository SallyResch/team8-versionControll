Git Changes

If you have done some changes to one file or different files you need to add it to the remote repository on Github. To do that you need to first check what changes you made and if you want to keep every single change or just want to keep one change.

Git commands

git checkout -b nameOfNewBranch
git add .
git add filename.html
git commit -m "message"
git push
